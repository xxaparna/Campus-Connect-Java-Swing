CampusConnect is a desktop-based Java Swing application for campus event management and user administration.

## Features
- User authentication (Login / Signup)
- Event creation and management
- Admin & User panels
- File-based persistence

## How to Run
1. Open project in any Java IDE
2. Ensure JDK 8+
3. Create empty `users.txt` and `events.txt` files
4. Run `CampusConnectGUI.java`
